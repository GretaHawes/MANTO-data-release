This is a release of the most recent publication of the data behind MANTO's public interface, available at [https://manto.unh.edu/publication.s/2616/](https://manto.unh.edu/publication.s/2616/). MANTO is a dynamic digital portal of Greek myth. It allows anyone to access, explore, and understand a storytelling tradition that extends over 2500 years. It provides authoritative data for researchers that will make big questions about the dynamics of myth answerable at unprecedented scale.
