## Description
This is a release of the most recent publication of the data behind MANTO's public interface, available at [https://manto.unh.edu/publication.s/2616/](https://manto.unh.edu/publication.s/2616/). MANTO it a dataset that models the Greek mythic storyworld and its impacts on the historical landscape of the Mediterranean using evidence from ancient sources. It provides authoritative data for researchers that will make big questions about the dynamics of greek myth answerable at unprecedented scale.

## Data
A .ZIP file of the public MANTO data is released on GitHub [https://github.com/GretaHawes/MANTO-data-release](https://github.com/GretaHawes/MANTO-data-release) and Zenodo [DOI](DOI) with records of all previous versions. The .ZIP includes .CSV and .JSON files of public [Objects](https://nodegoat.net/documentation.s/52/object-type) and [Classifications](https://nodegoat.net/documentation.p/450.m/53/classification). If you would like to download select Objects and Classifications (i.e., ‘tables’), then .CSV and .JSON files can be downloaded from the most recent release on NodeGoat, accessible at [https://manto.unh.edu/publication.s/2616/](https://manto.unh.edu/publication.s/2616/). 

The data model is visualised and described at [https://manto.unh.edu/publication.s/2616/](https://manto.unh.edu/publication.s/2616/). If one wishes to understand more about the database, its creation and organisation, and the data captured by the various fields, please see our manual at [https://www.manto-myth.org/documentation](https://www.manto-myth.org/documentation).
